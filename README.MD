# ModDril Modular System

A lightweight Laravel modular domain system for building scalable, domain-driven applications.


---

## 🚀 Introduction

ModDril Modular System provides a simple and powerful way to structure Laravel applications using a Domain-based modular architecture.

Each module:

- Has its own Service Provider
- Loads its own routes
- Can be enabled/disabled
- Is fully isolated

Perfect for large applications, SaaS systems, and domain-driven design.

---

composer require moddril/modularsystem
